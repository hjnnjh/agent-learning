# Communication Language

1. **Respond in Simplified Chinese (简体中文)** — this is a personal study repo.
2. **Tone**: professional, concise, direct. State facts and tradeoffs; avoid filler.
3. Keep code identifiers, file paths, URLs, math notation, and tensor shapes verbatim.
4. When explaining a concept for learning, prefer "what it is → why it matters →
   key takeaway", and link the relevant resource in the stage `README.md` rather
   than inventing a URL.
